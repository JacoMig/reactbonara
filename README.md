# Test Come si fa la carbonara
### Node version 16.13.2 
#### Stack
* React w/ Typescript
* eslint
* basic css
#### How to install
* git clone git@github.com:JacoMig/reactbonara.git
* cd reactbonara
* npm i
* npm run start (dev)
* npm run build (prod)
* npm run lint (linter)
* npm run test (runs test)
